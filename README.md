# Ames House Price Prediction

A team project using statistical learning and machine learning methods to predict housing prices in Ames, Iowa.

## Project Overview
We analyzed the Ames housing dataset and built predictive models to estimate sale prices based on property characteristics. The project included data cleaning, feature engineering, model comparison, and performance evaluation.

## Methods
- Exploratory Data Analysis
- Feature Engineering
- Linear Regression
- LASSO / Regularization
- Random Forest
- Gradient Boosting / XGBoost

## Results
Our final model improved prediction accuracy and reduced valuation error, especially for higher-value homes.
